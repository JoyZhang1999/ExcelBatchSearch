# Excel数据处理工具 - 使用说明

## 工具简介

Excel数据处理工具是一个基于浏览器的Web应用程序，允许用户上传两个Excel文件（条件文件和数据文件），根据用户指定的列号进行数据匹配，并导出匹配结果。

## 功能特点

- **双文件处理**：支持条件文件和数据文件分别上传
- **灵活配置**：可自定义选择用于匹配的列号
- **实时预览**：处理完成后可在线查看匹配结果
- **一键导出**：支持将匹配结果导出为Excel文件
- **响应式设计**：适配各种屏幕尺寸的设备

## 使用步骤

### 1. 上传文件

- **条件Excel文件**：包含查询条件和匹配规则的文件
- **数据Excel文件**：包含需要筛选的原始数据的文件

### 2. 配置筛选条件

在"筛选配置"区域输入要作为筛选条件的列号：
- 列号从1开始计数（例如：1表示第一列，2表示第二列）
- 默认值为2，即使用第二列作为匹配条件

### 3. 处理数据

点击"处理数据"按钮，系统将：
- 解析条件文件中的查询条件
- 根据指定列号从数据文件中筛选匹配的记录
- 在页面中显示匹配结果

### 4. 下载结果

点击"下载结果Excel"按钮，将匹配结果导出为Excel文件。

## 技术说明

### 文件格式支持
- 支持 `.xlsx` 和 `.xls` 格式的Excel文件
- 使用第一个工作表进行处理

### 数据处理逻辑
1. 条件文件的第一行作为表头，后续行作为匹配条件
2. 数据文件的第一行作为表头，后续行作为数据记录
3. 系统会检查数据文件中指定列的值是否出现在条件文件的任何列中
4. 所有匹配的记录会被提取并显示

### 浏览器兼容性
- 支持所有现代浏览器（Chrome、Firefox、Safari、Edge等）
- 需要启用JavaScript

## 注意事项

- 确保条件文件和数据文件使用相同的编码格式
- 处理大型文件时可能需要较长时间，请耐心等待
- 匹配过程区分大小写
- 建议先使用小规模数据进行测试

## 故障排除

### 常见问题

1. **文件无法上传**
   - 检查文件格式是否为.xlsx或.xls
   - 检查文件是否损坏

2. **处理按钮不可用**
   - 确保已同时上传条件文件和数据文件

3. **没有找到匹配数据**
   - 检查条件文件和数据文件的内容
   - 确认指定的列号是否正确
   - 检查数据格式是否一致

4. **处理过程出错**
   - 检查浏览器控制台是否有错误信息
   - 尝试刷新页面重新上传文件

## 隐私与安全

- 所有文件处理均在浏览器本地完成
- 数据不会上传到任何服务器
- 关闭页面后所有数据将被清除
